# How to Deploy

This project is a static website (HTML, CSS, JavaScript) with no build step required.

## Option 1: Run Locally
Simply open `index.html` in your web browser.

## Option 2: GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Source**, select `Deploy from a branch`.
4. Select `main` (or `master`) and `/ (root)`.
5. Click **Save**.

## Option 3: Netlify / Vercel
- **Netlify**: Log in and drag/drop the project folder to deploy.
- **Vercel**: Import the git repository. Select "Static" or "Other" as the framework preset (it should detect HTML automatically).
